# Gestión de consumo en comedor universitario

La universidad quiere analizar el consumo de comidas de los estudiantes durante una jornada.

Construya un programa que permita:

1. Solicitar cuántos tipos de comida se ofrecerán.
2. Crear arreglos para almacenar:
   - nombre de la comida
   - tipo
   - cantidad disponible inicial
   - cantidad consumida
3. Registrar la información de las comidas.
4. Solicitar la cantidad de estudiantes que asistirán al comedor.
5. Por cada estudiante:
   - solicitar nombre
   - solicitar la comida deseada
6. Verificar si la comida existe.
7. Verificar si aún hay unidades disponibles.
8. Si hay disponibilidad:
   - descontar una unidad disponible
   - aumentar el contador de consumo
9. Si no hay disponibilidad:
   - informar que la comida está agotada

### Al finalizar, mostrar:
- cantidad restante de cada comida
- cantidad consumida de cada comida
- total general de comidas servidas
- la comida con mayor consumo
- la comida con menor consumo

### Lo que se evaluará
- Uso de arreglos paralelos
- Actualización de datos
- Cálculo de máximos y mínimos
- Uso de ciclos y condicionales
- Claridad en la solución
